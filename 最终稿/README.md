# 最终稿

本文件夹用于存放确认后的毕业设计说明书最终稿内容。

## 存放规则

后续用户确认好的论文内容，将按目录结构一节一节放入本文件夹。建议采用以下命名方式：

- `00_目录.md`
- `01_摘要.md`
- `02_Abstract.md`
- `03_引言.md`
- `04_第一章_选题背景与研究意义.md`
- `05_第二章_设计方案与构思.md`
- `06_第三章_数据整理与信息架构.md`
- `07_第四章_设计实践与视觉呈现.md`
- `08_第五章_延展设计与应用.md`
- `09_结论与展望.md`
- `10_参考文献.md`
- `11_附录.md`

具体文件名以后续上传目录为准。每一节内容写入前，先以用户最终确认文本为依据，不擅自改动论文方向、章节编号和核心数据。
