# 南京艺术学院美术馆合作指数量化方案

## 📋 核心理念

合作指数（Cooperation Index, CI）用来衡量每一年南艺美术馆与外界合作的**广度、深度、国际化程度**。

---

## 🎯 方案设计

### 一、合作指数构成（三维度模型）

合作指数 = **机构多样性 × 合作深度 × 国际化权重**

#### 1️⃣ **机构多样性指数（Diversity Index, DI）**
- 反映该年合作机构的数量和丰富度
- 公式：DI = 该年合作机构数 / 历年平均机构数

**意义**：
- DI > 1.0：合作机构多于平均水平（高度多样化）
- DI = 1.0：合作机构等于平均水平（正常水平）
- DI < 1.0：合作机构少于平均水平（集中化）

---

#### 2️⃣ **合作深度指数（Depth Index, DPI）**
- 反映合作质量和频繁度
- 公式：DPI = 该年合作记录数 / 该年合作机构数

**意义**：
- DPI > 2.5：深度合作（平均每家机构超过2次合作）
- DPI = 2.0 ～ 2.5：中等合作深度
- DPI < 2.0：浅层合作（平均每家机构不足2次合作）

**解读**：该指数越高，说明与少数机构合作更深入；越低则合作相对分散。

---

#### 3️⃣ **国际化权重指数（Internationalization Index, II）**
- 反映国际合作的比重和国际影响力
- 公式：II = (国际合作记录数 × 2 + 全国合作记录数 × 1) / 总记录数

**说明**：
- 国际合作计权重 2.0（最国际化）
- 全国合作计权重 1.0（中等开放度）
- 地区合作计权重 0.5（本地化）

**范围**：0 ～ 2.0
- II ≥ 1.5：高度国际化
- II = 1.0 ～ 1.5：中等国际化
- II < 1.0：本地化倾向

---

### 二、综合合作指数计算

#### **模型A：平衡权重模型**（推荐）

$$CI_A = (DI × 0.3) + (DPI × 0.2) + (II × 0.5)$$

**权重说明**：
- 国际化权重 **50%**（最重要，体现开放度）
- 机构多样性 **30%**（重要，体现合作广度）
- 合作深度 **20%**（参考，体现持续性）

**指数范围**：0 ～ 2.5
- **CI > 1.5**：优秀合作年份
- **CI = 1.0 ～ 1.5**：良好合作年份
- **CI < 1.0**：合作有待加强

---

#### **模型B：国际优先模型**（侧重国际化）

$$CI_B = (DI × 0.2) + (DPI × 0.2) + (II × 0.6)$$

**权重说明**：
- 国际化权重 **60%**（优先目标）
- 机构多样性 **20%**
- 合作深度 **20%**

**适用情况**：如果美术馆战略重点是国际交流与合作。

---

#### **模型C：规模驱动模型**（强调合作广度）

$$CI_C = (DI × 0.4) + (DPI × 0.3) + (II × 0.3)$$

**权重说明**：
- 机构多样性 **40%**（最重要）
- 合作深度 **30%**
- 国际化权重 **30%**

**适用情况**：如果美术馆追求与更多机构建立合作关系。

## 🔍 指数解读框架

### 合作指数的四象限分析

```
高国际化 │  象限II        │  象限I
        │ (高国际+低广度)  │ (高国际+高广度)
────────┼─────────────────┼──────────
低国际化 │  象限III       │  象限IV
        │ (低国际+低广度)  │ (低国际+高广度)
        └─────────────────┘
           低机构多样性  高机构多样性
```

- **象限I**：理想状态 → 国际合作多，机构多样
- **象限II**：国际深耕 → 与少数国际机构深度合作
- **象限III**：合作薄弱 → 需要加强合作力度
- **象限IV**：本地开放 → 本地合作多但国际化不足

## 💡 建议的应用方式

### 第一步：选择合适的模型
- **重视国际影响** → 选择模型B
- **追求合作平衡** → 选择模型A（推荐）
- **强调合作规模** → 选择模型C

### 第二步：逐年计算
根据每年的合作机构数据，计算DI、DPI、II，然后代入公式。

### 第三步：趋势分析
- 绘制合作指数的年度折线图
- 分析增长、下降或波动的原因
- 对标行业标准或历史平均值

### 第四步：决策支持
- 识别合作高峰年和低谷年
- 评估合作策略的有效性
- 制定未来合作目标
