# 数字美术馆3.0 数据清洗与修复报告

**目标数据集**：历史文本数据（共计 3461 篇 Markdown 文章）  
**报告说明**：本报告记录了对全量 Markdown 文本数据进行深度清洗、排版规整及元数据结构化的全过程。

---

## 一、 数据重组与物理合并

- **任务说明**：原始目录下存在 3461 个散落的 `.md` 文件，管理成本高，且单文件过大导致常规 Markdown 预览器截断。
- **处理策略**：按文件编号升序提取，进行物理均分合并。
- **执行结果**：将 3461 篇文章均匀合并并重新打包为 50 个独立的聚合文档（`merged_1.md` 至 `merged_50.md`），每份约含 70 篇文章。不仅解决了单文件过长的问题，也为后续的批处理脚本提供了稳定的结构基础。

## 二、 脏数据与排版清洗

针对抓取过程中遗留的排版乱码及冗余占位符进行精准净化：

### 1. 清理无效排版占位符
- **任务说明**：大量文章中遗留了单独成行的感叹号 `!`。
- **处理策略**：通过正则匹配完全由单独感叹号构成的行进行剔除。
- **执行结果**：从 50 份合并文档中共计移除了 **34,244** 处无意义占位符。

### 2. 清理失效图片链接与冗余网址
- **任务说明**：正文中存在由于微信图文转换带来的冗长且失效的图片代码（如 `![](http...)`）及散落的长串 URL。
- **处理策略**：使用正则表达式 `!\[.*?\]\(.*?\)` 及 URL 特征码进行全局匹配清洗。
- **执行结果**：共去除 **902** 处 Markdown 图片占位符，并移除了 **375** 处裸露的 URL，极大提升了纯文本阅读体验及 NLP 纯净度。

### 3. 数据区异常词条排查
- **任务说明**：第一轮报告曾指出存在 8 个误入数据区的异常词条行（如“心理学讲座”、“肢体剧场”等）。
- **处理策略**：在最新构建的 50 份合并文档中进行全局检索。
- **执行结果**：未命中任何异常词条，确认当前数据区已无此干扰。

## 三、 数据去重

- **任务说明**：历史发布记录中存在针对同一事件的多次重复推送（如反复推送的“展讯推荐”、“闭馆通知”等）。
- **处理策略**：提取每篇文章的真实标题，建立全局哈希表进行精确匹配去重。
- **执行结果**：识别并删除了 **127** 篇高度重复的内容，去重后保留了 **3,334** 篇高质量的唯一文章。

## 四、 元数据校验与修复

### 1. 基础发文日期校验
- **任务说明**：核对文件名携带的日期是否与正文元数据区（如 `>作者： 时间：`）的日期发生冲突。
- **执行结果**：提取并逐一比对后，确认全部文章内部时间均与文件编号提取时间完全一致，数据一致性良好。

### 2. 活动时间区间格式规范化
- **任务说明**：正文中的“活动/展览时间”存在大量缩写格式（如 `2014.09.19—11.30` 或 `2019.10.11-20`），易导致下游解析系统抛出“结束时间早于开始时间”的逻辑错误。
- **处理策略**：编写脚本自动识别省略年份/月份的区间，提取前半段的时间特征智能补全给后半段，并兼容跨年判定。
- **执行结果**：在 35 份文档中成功修复补全了 **530** 处缩写的日期区间。

## 五、 结构化分类标签注入

- **任务说明**：原始数据缺乏统一的内容分类维度。
- **处理策略**：从标题或首行特征中提取原始标签（如【今日展览】、`丨AMNUA现场`），并编写映射字典，将其统一归类为 6 大标准维度：展览类、公教类、资讯类、学术与评论类、现场与记录类、其它服务类。
- **执行结果**：生成了新的 `>统一分类：xxx` 和 `>原始标签：xxx` 字段并注入至各文章元数据区，存入了名为 `categorized` 的专用副本目录下，为后续统计奠定基础。

---

## 六、 结论与下一步建议

**清洗结论**：
历经重组、去重、去噪、时间修复与维度对齐后，当前包含 3,334 篇文章的数据集已达到高度“干净”且“结构化半成品”的标准。

**下一步建议（Next Steps）**：
1. **数据矩阵化抽取**：鉴于元数据格式（时间、标题、统一分类、正文）已被高度规范化，建议编写提取脚本，将这 50 份 Markdown 文本转化为结构化的 CSV/Excel 表格，以接入数据库或 BI 可视化工具。
2. **正文 NLP 分析**：可对纯净的正文文本开展词频分析（TF-IDF）或主题聚类（LDA），挖掘高频艺术家与历年核心议题。
3. **异常数值清理**：后续建议清洗诸如 `评论：-100` 等抓取失效带来的异常互动数值。
