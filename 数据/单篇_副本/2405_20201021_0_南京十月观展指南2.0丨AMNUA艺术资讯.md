**南京艺术学院美术馆 AMNUA**南京市鼓楼区虎踞北路15号

**南京艺术学院2020**2020/09/28 - 10/25免费（目前仅对南艺师生开放）!!南京艺术学院美术馆于2020年9月28日——10月25日重磅推出《南京艺术学院2020：师生作品联展》。本次展览展出作品包括南京艺术学院美术学院、设计学院、舞蹈学院、传媒学院、流行音乐学院、人文学院、工业设计学院、高职成教学院、文化产业学院师生作品，在南京艺术学院美术馆1、2、3展厅展出。

**四方当代美术馆**南京浦口区珍七路9号

**四方特别展映项目｜我们在此：未来态**2020/10/15 -11/15门票/80元!!四方当代美术馆很荣幸为大家带来“未来态 （Future Ecologies）”特别展映项⽬——英国⽂化教育协会（British Council）与LUX机构策划“我们在此：英国⽂化教育协会以及LUX机构艺术家影像展”系列展映项⽬中的⼀个单元。本项⽬由包括2018年特纳奖得主夏洛特·普罗哲在内的四位长期驻扎在英国的年轻艺术家的影像作品构成，这些作品主题涉及古老医术与现代医疗体系之神话与现实、后殖民主义语境中的废弃电器“采掘业”、艺术家随机拍摄的LBGTQI群体图景及化身多座未来建筑的个性独特。四部作品将在四方当代艺术湖区会议中心主放映厅向观众呈现。

**金茂汇**南京市鼓楼区中央路201号 金茂汇商场一楼SHOW SPACE

**引力法则**2020/10/18 - 11/30免费!!“引力”于我们无处不在，它是连接每一个生命体征的隐形纽带，也是向宇宙与念想发出的共鸣。在引力法则之中，我们不再拘泥于现实的意义，而去感受被这种无形纽带环抱的一种温情，让所念所想，更好的完成自己。本次展览由艺厘米发出邀请，陈陈陈作为策展人与艺术家葛宇路、能尖日、Kanes&安阳&3ASiC、沈凌昊、李论、郝经芳&王令杰，共同在南京金茂汇创造出一个“引力法则”的场域空间，也希望这样的空间载体向所有年轻人发出引力的感召。不是所有的引力都能被找寻，或得到回音，但我们尽量让它纯粹的汇聚于此。

**问象艺术空间**南京石头城路69号留创园3号楼 A展厅,1号楼203室 B展厅

**甜蜜蜜:马焘个展**2020/10/17 - 11/09A展馆免费!!问象艺术空间将于2020年10月17日至11月09日

为您呈现艺术家马焘的最新个展《甜蜜蜜》展览将带来艺术家马焘新的油画作品.

**广延：魏志成个展**2020/10/17 - 11/02B展馆免费!!看到魏志成的这一组画，很难不让人想到弗朗西斯･培根的画：遒劲的轮廓，明朗的结构。特别之处是魏志成的县城经验。这些类似某所县城初级中学的场域经验，让魏志成在处理他那些画面形象时，有了更为明确的形象化的意图。用魏志成自己的话说，就是“我开始关注一些奇怪的建筑结构和畸形的几何形体，一种几何化的结构图形出现在画面中用来表达一种陌生化的情境。……它可以传达出一种冷漠的不带情感色彩的视觉效果，这种冷峻硬朗的画面质感可以表达一种哲学化的内心感受。”这些几何形体和人体所形成的意象交合，让魏志成的画面奇迹般地化合而成一种“卡夫卡”式的诡谲气氛，或许这就是魏志成所追求的画面“叙事性”。

**和光陶社**南京市鼓楼区宁夏路1-1号左侧院内

**采 采 \-\- 齐冬根瓷画展**2020/10/06 - 10/21免费!!老齐不是个“专业”画家，他有着丰富的工作经历，我所知道的有：农民、打工仔、司机、录像厅和民宿的老板，画家只是众多身份之一，既不以画谋生，也不在甚么“院所”，故不称作“专业”画家。画画是自学的，无门无派，亦中亦西，“我是很真实的”，只要你有兴趣，他就愿意把他的经历分享，一提及他的画，老齐就有了低调的得意，他可以一张张地介绍作品，画中人物是谁，又为什么画他/她，最关键的是，他有着自己的绘画理念和坚守。既然没有老师，不入流派，同时画种也算独特——瓷板之上的釉画，就免于绘画的鄙视链条之中，我亲耳听一位国画届的高人，在本市的某美术馆大言不惭：我们国画的线条是最高级的。嗐！老齐大不了画点杯子，即便放在网上售卖，也颇有销路的，做买卖还不用陪酒陪笑说大话，此般自由，正是我辈所求而不得的。

**南视觉美术馆**南京市鼓楼区中山北路30号43层

**匕首轻哼**2020/10/11  - 10/25门票 / 以场馆实际为准!!!!与其感慨南京的文化和商业环境下，还剩多少做“当代”的年轻艺术家，不如看看本地青年们正以哪些文化生产来回应当下。本次展览试着呈现其中，一些敏锐散漫，将日常经验转化出材质与形式的潜流。他们是省城的艺术教师、熬夜社畜、机构员工、餐厅摄影、待业诗人、友邻结社，不善酒局与口号，却精通散步走神，在自我生活、劳动、消费、景观的夹缝里就地取材，打磨起个体化的（武器/玩具）。暗室中装置、影像、窗口的微光，如匕首扎破现实，又似止于半嘲半解的轻哼。危、机四伏的紧缩时代，呼唤起对附近的回归。这是微观反抗、具体团结的新演练？还是解体后故作轻盈的徘徊？结论或许尚早。反正来路莽莽、去路茫茫，迫不及待的行动恰是最好的行动。
