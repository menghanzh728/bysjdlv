南京艺术学院美术馆

从即日起开放预约

校外观众也可进校观展啦！

为了给广大观众提供便捷的观展途径

现将预约观展方式整理如下：

**预约须知**

1

// **预约说明**

请各位观众计划好出行时间，**提前一天预约进校。**

**预约时间：** 上午9：00-12：00               下午14：00-17：30

预约生成的二维码 **只能进出一次，**且32小时内有效

**美术馆开放时间：** 9：00-16：30**（逢周一闭馆）**

**预约人数：**约满即止

2

// **预约方法**

**第一步：**关注 **“平安南艺”** 公众号在 **“扫码入校”** 栏目中选择 **“访客预约”**!

**第二步：**在来访登记中将带 **\*** 的栏目填写清楚**受访者姓名填“参观展览”****信息漏填的观众无法被核准通过**!

**第三步：**点击提交访客信息您将看到以下画面接下来就是耐心等待后台核实并处理通过!

**第四步：**如果核实您的信息无误并在通过预约的 **最高人数以内**您将收到来访通过提醒如下：!

**第五步：****进校前请戴好口罩****准备好健康码****出示预约进校二维码**!

**为保障常态化疫情防控期间开馆需要，确保观众进馆观展安全，请您严格遵守以下须知：**一、本馆实行总量控制，超过入馆控制人数后停止进馆。二、开馆时间：周二至周日9：00-16：30接待观众，16：30开始美术馆整体消毒。三、为保证观众在美术馆观展期间活动轨迹留存，预约进校访客信息填写必须真实有效，否则不予通过。四、访客须凭身份证、健康码、预约码进校，全程佩戴口罩，保持安全社交距离进馆、离馆。五、美术馆展期结束将暂停本次预约通道。
