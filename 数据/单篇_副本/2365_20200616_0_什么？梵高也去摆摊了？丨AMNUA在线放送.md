# 关于插画师

Alireza Karimi Moghaddam，是一位伊朗的插画家、漫画家，现居里斯本。

十几岁的时候，因为爱上了梵高的人生故事和绘画风格。从此追随所有关于梵高的书籍和电影，沉浸其中。

插画家在采访中说道：梵高不仅是一位画家和天才艺术家，更是一种生活方式。他不是一个痛苦或失意的榜样，而是充满爱与关怀、人性善良和利他主义者的代表。

插画家Karimi 希望从不同的角度去解读梵高的生活，想让不同的梵高成为每个人心中的梵高。
