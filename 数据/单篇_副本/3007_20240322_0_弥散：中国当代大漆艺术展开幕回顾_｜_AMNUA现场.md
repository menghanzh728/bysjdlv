# 3月20日，由南京艺术学院主办，南京艺术学院美术馆承办，福建省美术馆、厦门大学艺术研究所、南京艺术学院设计学院、华东师范大学中国当代漆艺术中心协办的展览《弥散：中国当代大漆艺术展》，在南京艺术学院美术馆开幕。

!

江苏省政协原副主席、南京艺术学院原校长、中国美术家协会漆画艺委会名誉主任冯健亲教授，厦门大学艺术学院院长刘赦教授，中国美术家协会漆画艺委会副主任、南昌大学王向阳教授，中国美术家协会漆画艺委会副主任兼秘书长、厦门大学汤志义教授，南京师范大学美术学院院长刘玉龙教授、美术学院吴可人教授，福建省美术馆馆长、福建师范大学硕士生导师邱志军教授，华东师范大学美术学院马俊营副教授；南京艺术学院党委书记俞锋教授，校长张凌浩教授，艺创处处长金捷教授，美术学院院长章文浩教授，设计学院党委书记张健、院长蒋杰教授出席开幕式；设计学院领导班子及来自各界代表近二百人参加开幕式。

!江苏省政协原副主席、南京艺术学院原校长、中国美术家协会漆画艺委会名誉主任冯健亲教授致辞

冯健亲介绍了南艺漆艺专业的历史，他提到，南艺漆艺专业在全国艺术高校领域具有较高的学术水准。在谈及漆艺术的发展时，冯健亲指出，现代漆艺、漆画以及传统的漆器制作，实际上并不是某种演进或替代的关系，而是漆文化在不同历史阶段发展的产物，并在如今共同成为漆文化体系中的重要部分。需要关注的是，现代漆艺作为三者当中与当下社会、时代背景关系极其密切的一种，无论它涉及怎样的文化命题，作为艺术形式的一种，它所具有的价值终将通过艺术的标准加以评判和展现。

!南京艺术学院校长张凌浩教授致辞

张凌浩指出，漆艺术在漫长的发展历程中积淀下深厚的审美底蕴和价值。南京艺术学院工艺美术专业具有悠久的办学历史，其起源可以追溯到建于1912年的上海美术专科学校设立的工艺图案科，1986年，南艺成立漆艺专业。发展至今，漆艺专业在科研和教学上建设不断，所在工艺美术专业获批国家本科专业建设点；《漆材料与工艺》获国家级一流本科课程。因此，南艺主办本次《弥散——中国当代大漆艺术展》的初衷，正是基于当代艺术视角中的漆艺术涉猎范畴领域扩大化的趋势。未来，南艺漆艺仍将谨遵“闳约深美、不息变动”校训，以建设“中国工艺美术大师传承创新基地”和“江苏省中华优秀传统文化漆艺术传承基地”为契机，以实验性教学为主体的教学理念推进专业建设，为中国漆艺术发展提供南艺路径。

!艺术家代表、中国美术家协会漆画艺委会副主任、南昌大学王向阳教授致辞

王向阳表示，“弥散”作为本次展览的主题，不仅是一个艺术概念，更是一种表达自我、探索世界的方式。在他的创作旅程中，始终致力于通过艺术探索生命的本质，表达内心最深处的情感和思考，尝试捕捉在日常生活中易于被忽视的瞬间，以及那些在宏大叙事之外的微妙情感。正如展览《弥散》所暗示的，艺术在其细腻、几乎无形的存在中，触及了人类共有的情感和经验。大漆艺术的美妙之处，在于它既是极其个人化的表达，也是跨越时间和空间的共鸣。在《弥散》中，不仅展示了艺术作品，更希望能够搭建起一个平台，让思想和感情在这里自由流动，让不同的声音被听见，不同的视角被看到。

!南京艺术学院美术馆副馆长林书传致辞

林书传谈到了大漆的历史底蕴和独特的艺术魅力。他认为，漆画经过世代的传承与发展，早已不仅是一种绘画技法，更是一种文化、一种精神的象征。在当代，漆画艺术迸发出前所未有的生机与活力。从此次展览作品中，可以看到艺术的多样性与包容性。有的作品以山水为题材，运用丰富的色彩与细腻的笔触，将大自然的壮美与灵动展现得淋漓尽致；有的作品以人物为主题，通过细腻的描绘与传神的表情，展现了人性的丰富与多彩；还有的作品以抽象为形式，大胆运用色彩与线条的组合，表达了艺术家对世界的独特理解与感悟，用漆的语言，诉说着漆画艺术的辉煌历程与美好未来。林书传指出，此次展览不仅是一次艺术的盛宴、一次文化的交流，更是一次心灵的洗礼、一次精神的升华。

!中国美术家协会漆画艺委会副主任、南京艺术学院李永清教授主持开幕式

!

冯健亲、张凌浩、刘赦、汤志义、邱志军、马俊营参加展览揭幕仪式。本次展览将持续至4月7日。
