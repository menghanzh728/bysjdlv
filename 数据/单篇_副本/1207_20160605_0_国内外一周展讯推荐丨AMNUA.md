# 王后·母亲·女将 — 纪念殷墟妇好墓考古发掘四十周年特展

展厅

一层B展厅

费用

Free（需提前一天预约）

妇好墓，即殷墟五号墓，1976年在河南省安阳市殷墟发掘出土。墓主为商王武丁的王妃「妇好」，中国历史上有文字记载的最早的女政治家、军事家。妇好墓共出土1928件随葬器物，是殷墟唯一保存完整的王室墓葬，也是目前唯一能够跟历史文献和甲骨文联系，进而推定具体墓主的殷代墓葬，有着重要价值。

本次展览共展出411件妇好墓出土的文物，包括青铜器、玉石宝石器、骨器、陶器四大类；展览内容分为序厅、四个单元及尾声，共六个部分。

!

周五、周六夜场延迟至24:00 周一闭馆

阿尔伯托·贾科梅蒂（Alberto Giacometti，1901年10月10日－1966年1月1日），瑞士雕塑家、画家。贾科梅蒂以超现实主义、立体主义的雕塑作品闻名，他的作品对战后现代艺术的发展产生了深远的影响，被誉为现代主义精神的化身。

本次展览将展出贾科梅蒂的约250件作品，其中包括最具标志性的《行走的男人》（1960）、《勺型女人》 （1927）、《鼻子》（1947）、《笼》（1951）等，涵盖了艺术家从早年创作，历经立体主义、超现实主义到1960年代多产杰作期等不同阶段。此外还将展出贾科梅蒂的肖像素描、绘画、摄影、原始的石膏、青铜雕塑以及手稿等档案资料。

本次展览是贾科梅蒂在中国的首次大展，也是继2007年法国蓬皮杜艺术中心之后全球最大规模的回顾展。展览总策展人为贾科梅蒂基金会总监凯瑟琳·格雷妮尔（Catherine Grenier）。

黄永砅，1954年出生于福建厦门，1982年毕业于浙江美术学院，1989年开始工作及居住在法国。黄永砅持虚无主义倾向，强调怀疑论的重要性，反对理性主义以及以人类为中心的霸权主义，探讨中西文化之间的关系，试图寻找一种可以超越国界以及意识形态冲突的表达方式。

展览将展出黄永砅近30件作品，除了《蛇杖》等数件大型装置外，更回顾性地重访了其大量的早期作品，试图展现艺术家多年来持续的探索、反思和创作。同时，亦希望呈现中西方的交汇之处，并由此引发关于文化认同议题的讨论，挑战意识形态与制度的推测；最终该项目将为当代地缘政治的真实情况提供一种批判性的分析。

本次展览将展出艺术家罗必武和钟曦双的油画及版画作品，呈现两位艺术家在绘画方面的创作与革新。

罗必武的作品以「戏仿」的手法将一些经典符号和新闻图片植入或转印到画面及其他媒介上，使公共事件与私密空间，宏观与微观交织在一起，形成某种玄幻、错置的超现实氛围。「戏仿」是指以挪用和改造某些耳熟能详的形象或话语来达到嘲讽、游戏的效果。

除了「戏仿」的成分，罗必武还将人世的暴力、争夺、罪恶、堕落行为与人的原罪联系起来，与戏仿构成相互关联的两极，拓展了他创作的语言空间与精神力度。

广州红专厂当代艺术馆3号馆

费用

本次展览将展出4位以色列当代艺术家的作品，从不同角度诠释艺术家所感知的「中国制造」，参展艺术家包括玛雅·艾桐（Maya Attoun）、尼尔·哈雷尔（Nir Harel）、希拉托尼·纳科（Hila Toony Navok）和唐玛丽（Irit Tamari）。

「以色列制造」展览源自对中国的认知：跨越几千年的文化和地理，也是世界上最古老的文明之一，丰富多元的传统文化存在于各行各业，特别是在文学、诗歌和艺术上的精神和创意。

19世纪末至20世纪初，老子和孔子的著作被翻译成德语、英语和其他西方语言，以色列文化界也开始对老子的思想予以特别的关注，其著作先后6次被翻译成希伯来语。

过去几十年间，中国已在世界各地刻下了「中国制造」这一标志性的烙印。此次参展的4位艺术家将以个性化和感性的作品表述他们在中国的经历，并承载各自的艺术精神。

大都会艺术博物馆

费用

2016年5月，大都会艺术博物馆的服装研究部将推出一个新的时装展，主题围绕「手作 x 机器」，探索服装行业随新科技诞生而产生的变革，以及设计师在制作高定服饰或前卫成衣时，对传统手工艺与现代工业生产手段的复合运用。

展览将通过约100套服装，包括1880年代的沃斯（Worth）礼服至2015年的香奈儿套装，深入探索19世纪高定文化的形成，以及随着大众工业化生产的普及，手作与机械制作间的区别。

博物馆的 Robert Lehman 翼展厅将并置陈列手工制作的高定服饰和机器生产的成衣。另有一些展厅将模拟设计师工作室的传统格局，为观众展示刺绣、羽毛装饰、打褶、编织、花边、皮革、镶边等工艺，并介绍 3D 打印、雷射切割、汽蒸热定型、计算机制版、圆形罗纹针织、超声波熔接、压层等新技术。

考古学家于2001年在埃及亚历山大港附近的地中海海底发现了两座埃及古城 Canopus（卡诺珀斯）和 Thonis-Heracleion（伊拉克利翁），这两座古城大约建于公元前六、七世纪，距今2500年的法老王时期。

经过长达14年的多次打捞与残片组合，考古学家获得了多件珍贵文物，由于这两座古城在全盛时期吸收了来自希腊等地的外来文化影响，因此出水的文物中大多呈现出希腊文化。文物中的重要展品包括供奉土地神 Serapis、送子神伊希斯（Isis）、冥神 Osiris 等埃及宗教人物雕像；大量金币和来自雅典的石头砝码等。

# 印度艺术家 Bhupen Khakhar 回顾展

泰特现代美术馆

全票：10.9英镑 优惠票：9.5英镑

Bhupen Khakhar（1934-2003）出生于孟买的一个中产阶级家庭，原本在孟买大学学习文学、经济学和哲学，但在家人的坚持下，转而学习工商管理，毕业后成为一名注册会计师。

但他并未停止对艺术的追求，1960 年代早期 Bhupen Khakhar 开始自学绘画，并获得巴罗达美术学院的艺术批评硕士学位。1965年他举办了人生第一场个展，到1980年代他的名声已经传到伦敦、柏林、阿姆斯特丹和东京。

在 Bhupen Khakhar 的绘画中最常见到的是一些普通人，例如理发师、 钟表修理工，甚至是和他共事的助理会计师，也时常将印度的小商店置于画作中。此外，他还以公开的同性恋身份探索同志之间的爱、生活及在印度的遭遇，也以多情和色情的方式触及其文化内涵。

本次展览将展出世界各地收藏的 Bhupen Khakhar 的画作，回顾艺术家在印度艺术中发挥的重要作用。

# 时尚前沿：法国三百年时装展（1715-2015）

!

时间

2016年4月7日 - 8月14日 11:00 - 18:00

周三延长至21:00 周一闭馆

107, rue de Rivoli, 75001 Paris

* * *

为庆祝巴黎装饰博物馆「时尚馆」落成30周年，馆方策划了「时尚前沿：法国三百年时装展」，展出馆藏的300余件女装、男装和童装，呈现1715年 - 2015年间的法国时装艺术。

巴黎装饰博物馆「时尚馆」创建于1986年，馆藏时尚藏品15万件，包括古典纺织品、现代高定服装、经典成衣、饰物、以及 Madeleine Vionnet、Cristobal Balenciaga 等知名设计师的设计稿、作品影像等。

本次展览，策展方从藏品中精选300余件重要时装作品，以时间为序，回顾18世纪至今时装史上的重要时刻及经典作品，其中包括 Elsa Schiaparelli、Gabrielle Chanel、Paul Poiret 等人的设计作品。
