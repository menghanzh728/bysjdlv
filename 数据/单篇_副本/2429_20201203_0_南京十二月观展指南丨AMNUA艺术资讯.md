**博物馆 / 美术馆****Museum**

**南京艺术学院美术馆 AMNUA**南京市鼓楼区虎踞北路15号

**画刊 \-\- 45周年封面计划**2020/11/24 - 12/05免费（暂时仅对校内开放）!为纪念杂志创刊45周年，《画刊》在2019年推出了年度艺术项目“《画刊》45周年封面计划”，力图以每年一次项目的方式携手艺术家，以媒体生产的形式联动艺术创作，对中国当代艺术现场提出某种问题，并形成整体性的艺术出版与传播序列。

**档案：逐渐形成的闭环**

**2020中国当代水墨年鉴五周年特展**2020/11/19-12/19免费（暂时仅对校内开放）!“中国当代水墨年鉴”品牌项目从2016年启动，先后在北京今日美术馆、四川美术学院美术馆、广州美术学院大学城美术馆、中国美术馆、旧金山南海艺术中心等美术馆策划和实施，已经成功举办了五次大型展览。2020年，五周年之际，“中国当代水墨年鉴”来到了南京。

**程楠个人项目****Ramble Journey  Cheng Nan Individual Exhibition**2020/12/03 - 12/23免费（暂时仅对校内开放）!《漫行记》本是我创作的一套连环画名，后来决定用作此次项目的名称，也是比较合适的。在我看来，艺术创作是很个人的事情，究其本源，即是借由各种语言描绘日常生活点滴，抒发内心真实情感。——程楠

**亦安N空间**南京市玄武区估衣廊贰号贰零壹 XSPACE艺术空间

**迹 · 向 \| 罗可问个展**2020/11/25 - 12/15免费!!本次展览的主题为“迹·向 Trace / Towards”，是过去三年间艺术家罗可问对同一个模特的持续性创作。从初见、初画时的新鲜与神秘，至熟识、熟稔后笔触的明朗与清晰，再到反观、回望后的思索与沉淀……艺术家借被作画者之貌，还原自我之认知，使观者体会到画中人物微妙的显像状态与关系——一个能够深入受众内心的真实情感表象。

**金鹰美术馆**南京市建邺区应天大街888号A座52层

**如同我们，所有莫名的渴望 Ineffable Longings**2020/11/11 - 2021/02/18展厅 / 中央展厅门票 / 56元!!金鹰美术馆将于2020年11月11日,在中央展厅对公众开放“如同我们,所有莫名的渴望”年度大展项目,该项目由金鹰美术馆团队策展,将呈现六位艺术家的近期作品。从10月22日起,观众可在金鹰美术馆小程序上购票观展。

**全景：世界的因缘际会**

2020/09/26 - 2021/06/27展厅 / 广场门票 / 80元!!“全景：世界的因缘际会”是一个跨年度的项目，大体量的空间展示和城市景观遥相呼应。“全景”的概念最初来自于美术馆空间营造时的关键词之一，强调展览空间与都市场景的并置呈现，观众在展览空间和空中城市视野之间穿梭游走，艺术作品既是展览空间的一部分也是城市空间的一部分。建筑意义上的空间和艺术品构建的多时空、多地缘文化的空间交错，也对应了展览的副标题——世界的因缘际会。

**对流 With / Against the Flow**2020/09/26 - 12/06展厅 / 南展厅，Art Sight，小展厅门票 / 126元!!“对流 With/Against the Flow”是一个巡展项目，展览的标题强调中英文的对应，“对流”是展览策展团队的中方策展人陈瑞命名的标题，也是不同于巡展之前广州一站的中文标题的新的解读，这个再译的行动本身也诠释了文化交流中的错位解读和知识递增的状况。城市与人的关系是金鹰美术馆——作为一个城市社群新的成员所关注的话题，因此这个展览呈现出的命题也将是美术馆未来一以贯之的思考和探讨的方向。

**四方当代美术馆**南京浦口区珍七路9号

**灵与景 Ten Thousand Things**2020/11/08 - 2021/05/23门票 / 80元!!“灵与景”是艺术家米利亚姆·卡恩与克劳迪娅·马丁内斯·加拉伊之间，一场跨越世代、地域和文化的特殊邂逅。本次双个展呈现的作品涵盖油画、素描、水彩、装置、雕塑、影像、拼贴，以及为美术馆特定空间创作的壁画等丰富媒介。展览标题引喻古代宇宙观，如《易经》中所表达的——“天地交而万物通也，上下交而其志同也”。“灵”指向人类与万物众生，“景”则有关自然、环境和各种现代人造系统；“灵与景”从而诗意地将本次展览中的作品联系起来，凸显艺术家实践中的共通点，她们如何重思“灵”与“景”的关系，如何用艺术来表现这些关系。

**OCAT南京栖霞展区**南京市栖霞区欢乐大道与疏港大道交汇处

**必要的空--此岸：OCAT南京公共艺术计划·2020**2020/11/09 - 2021/03/07门票以场馆实际为准!!“必要的空”也可以理解为“留白”——在中国古代山水画中，留白不仅作为视觉空间的构成要素，同时也涉及到不可见的、更为形而上的宇宙观层面，将它引申到当今的艺术语境中，还可以联系着涉及艺术家在创作之中对于语言、媒介、技术和环境的怀疑和反观，由此本次展览也从主题性上与第一年度以水为元素的展览“流动的书”，第二年度以山为元素的“浮云的根”形成递进的结构，完整地对应了中国古代山水画的三个基本元素。

**陶谷公园**南京市秦淮区菱角市66号国创园9-2号楼1楼

**正负极 \-\- 吴博个人项目**2020/11/28 - 12/27免费!!TOKU POP-UP实验空间位于南京市秦淮区菱角市66号国创园9-2号楼的拐角，它以橱窗和快闪的形式进行当代艺术的展示。《正负极》是TOKU POP-UP的系列现场项目的开端，这个项目将致力于呈现艺术家在创作中对个体在群体中的相处模式的关注与思考。

**逸空间**南京市鼓楼区凤熙路27号

**在不远处**2020/11/07 - 12/09免费!!“在不远处”链接着世界的两端：此刻与未来。正如加缪所说：“对未来真正的慷慨是把一切献给现在”。这是一个不断回归与祛魅的过程，回归到自身与内心，也回归到最真实的生活本身，回归到家人与朋友之间，爱与健康也逐渐替代虚无缥缈的神话和光环，空旷的画室和空白的画布也消解这从前的喧嚣嘈杂和人声鼎沸。回归到此刻此时此地同时，我们也从未更加坚定地“相信未来”，无用的理想主义成为自我前行的动力因子，也成为构建乌托邦的坚实地基，寻找生活在别处的可能。

**幺幺七艺术空间**南京市秦淮区莫愁路329号越界梦幻城117栋B1

**Life-2：光明与黑暗**2020/11/21 - 2021/01/10免费!!逸空间荣幸地宣布，将联合艺品春山在幺幺七艺术空间共同呈现段革新的个人项目 Life-2:光明与黑暗。这是逸空间与艺术家段革新的第二次合作，Life-1的艺术项目正在南京二条商店展出。呈现的是艺术家对人与动物之间的关系的思考。Life-2:光明与黑暗，则是艺术家表达了对生命个体的四种状态的思考和理解，包含艺术家从2010年第一次个展至今的作品脉络以及最新的装置作品。

**肆间艺术中心**南京市建邺区河西大街239号 南京国际青年文化公园北侧

**“不同主線”——当代艺术展**2020/11/08 - 12/08免费!!这次“不同主线”当代艺术展以“经历”作为母题展开策划，旨在尝试突破当下的一种同质化现象，选择有着独特视角的艺术创作者，去呈现他们对于外界观察方式和体验方式的差异性，并同时避免当代艺术中不成系统、碎片化呈现的问题，做到一定的轴向梳理。--节选自王文文《当代艺术，从经历开始——写在“不同主线”当代艺术展前》活动详情>>>

**PUSPACE普世皮**南京市玄武区童卫路5号

**「本·源」孙睿 陶艺展**2020/10/31 - 12/09门票 / 30元 （凭学生证免票，需现场出示学生证） !!本次展览选择了孙睿从2017年至今的13组作品，是她在南京的首次个展。展览的主题将“本源”一词拆分开来，“本”代表着人性本质的终极议题，其中包含了生命、生活、情感、历史、技术等多个方面；而“源”则指向一种行为，是艺术家对以上哲学母题的追溯，也是观众观展期间所产生的思考。展览在“探索本源”这一主线之下分别从本体、情感和技术三个角度进行追问。由生至死、由熵到负熵、从无序到有序、从孩童到暮年……生命的饱和度不断增加，个体为之赋予更多的价值，我们的情感在进一步变得丰富和复杂，与此同时人类群体与技术间的关系也在不断地变化和发展。艺术家通过艺术的语言与观众形成对话，邀请大家一起暂时离开外界的浮躁和喧嚣，回溯本源，重回人类最初最为本真的模样。活动详情>>>

**竹园2020：杜荣坤个展**2020/11/28 - 12/13免费!!《竹园2020（时间、空间与意识的三重奏）》是杜荣坤的第一次个展，是他和南京的再次交集。展览的主题，于空间上取境故乡竹园，在时间上锁定当下，展览的内容贴合空间的分隔做布局，分而不离的三个空间各自对应展览的三大篇章：第一章时间、第二章空间、第三章意识。杜荣坤创作脉络中的关键词，依次为章，三个章节平行并置且相互交织，三重奏彼此呼应共鸣。

**南京美术馆**南京秦淮区常府街四条巷12号

**南京纬度（第二回）**2020/12/01 - 12/20免费!南京纬度第二回艺术作品展是一种愿望，我们试图努力在逐渐苏醒的当代艺术语境中寻求本土文脉的艺术延伸与新的探索或突围。每一个时代，能够留下的文化烙印，总是在后人看来属于那个时代的独特面貌和气质——回顾过去的十年、二十年，我们可以清晰地看到，中国当下的艺术现象从学习（拿来主义）西方现代主义，到分析、研究西方当代艺术，似乎得到这样一个结论——那就是认真对待自己，用心感知自身经验，寻求新的（有别于过去的）调性。这里面既有文化背景、文化自觉，又有经济社会的时尚潮流与新鲜活力。特别是80后、90后艺术现象蓬勃而出，更能够明显地感知到艺术“自我”觉醒的迫切与重要，并且通过不同区域呈现出昂扬的艺术情境与生态。

**FFA艺术中心**南京市江宁区龙眠大道668号龙庭水岸别墅区12-103

**那一些东西，恰似人生的行李**2020/11/12 - 12/13免费!!FFA ART CENTER即将于2020.11.19荣幸呈现新晋艺术家何镒的首个绘画个展，展览主题定义为“那一些东西，恰似人生的行李”，这一展览涉及艺术家从2017-2020年四个跨度的作品。艺术家借助日常的事物通过语言的建立，旨在提醒我们关注语言本身的视觉逻辑，以及物体本身与空间所带来的仪式感，在刀砍顿挫的凝练下，这些看似普通的物体向观众进一步揭示了语言的本质。

**南京博物院**南京市玄武区中山东路321号

**罗关根缂丝作品捐赠展**2020/09/01 - 待定展厅 / 艺术馆2楼临时展厅门票 / 以场馆实际为准!缂丝是一种中国传统丝织品，其工艺特色是以缂织方式织造，纬线并不贯穿全幅作品，而仅在需要处与经线交织，称为「通经断纬」法。2009年，缂丝织造工艺被列入世界非物质文化遗产。本次展览将展出苏州缂丝工艺美术师罗关根捐赠的《胤禛十二美人图》，结合院藏宋代至明清时期的部分缂丝作品，介绍缂丝的独特工艺，让人们直观感受传统工艺的魅力，呈现至善至美的工匠精神。

**PU SPACE 普世皮**南京市玄武区童卫路5号

**游 戏 之 作**2020/12/04 19:30门票/ 10元起!艺术起源的各种说法里，有一种是“游戏说”。康德认为艺术是”自由的游戏”，其本质特征就是无目的合目的性或自由的合目的性。席勒认为人的艺术活动是一种审美外观为对象的游戏冲动。席勒在康德的基础上更进一步，认为”过剩精力”是文艺与游戏产生的共同生理基础。艺术和游戏的关系密不可分，本次展览运用“游戏”这样一个看似轻盈的概念将青年艺术家的创作撕扯出来。
