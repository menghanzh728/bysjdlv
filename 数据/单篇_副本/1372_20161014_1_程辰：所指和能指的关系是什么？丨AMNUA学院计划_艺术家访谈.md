受访者：程辰（以下简称“程”）

!

程辰《red&green》

霓虹灯装置 Neon lights Install 120cm×83cm，2013

作品阐释：

《red&green》是霓虹灯装置作品，意在探讨和混淆概念和实体的边界，探讨所指和能指的关系，也是向观念艺术家科苏斯致敬。

Q：你是因为什么进入艺术圈的？

程：因为艺术。

Q：有没有什么因素可以对你的创作思路产生重大影响？

程：时间及其带来的一切。

!

程辰《red&green》

霓虹灯装置 Neon lights Install 120cm×83cm，2013

Q：作为学院计划的一员，学院的教育或者学院的环境给你带来什么样的影响？

程：美院给我持续而稳定的支持和包容。

Q：对于现在常出现的量子力学、生物遗传学等科学技术与艺术的跨界结合，你怎么看？

程：必将是一个全新的艺术世界和门类，非常期待且会持续关注，也希望有机缘参与。但在这样的技术浪潮下做作品，还是应该以艺术为核心来思考，不要被强大的资本和科技压倒，成为科普性的说明文。

!

程辰《red&green》

霓虹灯装置 Neon lights Install 120cm×83cm，2013

Q： 能否给我们简单介绍一下你本次参展的作品？

程：《red&green》是在探讨和混淆概念和实体的边界，探讨所指和能指的关系，也是向观念艺术家科苏斯致敬。

!

程辰《以前》Before

综合机械装置 Comprehensive machinery Install 400×150×200cm，2012

Q：你觉得学院展和非学院展的区别在于哪些方面？

程：系统、持续和严肃是学院展的特色。

Q：你觉得学院的环境对于艺术家来说是保驾护航还是温水煮青蛙？

程：兼而有之，关键看自己怎么做。

!

程辰《red&green》

霓虹灯装置 Neon lights Install 120cm×83cm，2013

Q：你觉得学院里有当代艺术吗？

程：当然，而且是中坚力量。
