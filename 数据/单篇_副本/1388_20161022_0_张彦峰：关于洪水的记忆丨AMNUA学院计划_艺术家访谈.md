张彦峰《关于洪水的记忆》The Memory of flood

多屏录像装置 5′7″，2015

作品阐释：

35年前的8月8日凌晨，从这个叫魏湾的村庄以上十多公里处，死神起步开始旅行：以下百多公里的华中平原，在数亿立方米的溃坝之水覆盖之下，数不清的人们瞬间毙命。

死神傲慢而坚定，在东西150公里，南北75公里范围内，杀戮所有的生物，人们被悉数淹没在高达十米的水舌之下——那是一个冰冷的夏天。

美国Discovery频道制作的“10toptechno-logical catastropheinthew orld”专题片，将此次溃坝事件列于首位，居于其后的是印度博帕尔化工厂泄毒事件和前苏联切尔诺贝利核电站爆炸事件。他们披露说，溃坝造成的直接死亡，加上后续因缺粮、感染、传染引起的死亡，人数共计24万多人。

Q：你是因为什么进入艺术圈的？

张：我一直不知道怎么界定我是不是在艺术圈里，如果非要说一个圈子的话我希望和“那种做不是让看起来昏昏欲睡作品”的人呆在一起！（借策展人的话）

!

Q：有没有什么因素可以对你的创作思路产生重大影响？

张：所处的时代吧。

Q：作为学院计划的一员，学院的教育或者学院的环境给你带来什么样的影响？

张：质疑与持续性思考。

!

张彦峰《关于洪水的记忆》The Memory of flood

多屏录像装置 5′7″，2015

Q：对于现在常出现的量子力学、生物遗传学等科学技术与艺术的跨界结合，你怎么看？

张：不明则厉。

!

张彦峰《关于洪水的记忆》The Memory of flood

多屏录像装置 5′7″，2015

Q： 能否给我们简单介绍一下你本次参展的作品？

张：1975年8月8日，特大暴雨引发的淮河上游大洪水使河南省驻马店地区包括两座大型水库在内的数十座水库漫顶垮坝，造成1100万亩农田受到毁灭性灾害，1100万人受灾，纵贯中国南北的京广线被冲毁102公里，中断行车18天，溃坝造成的直接死亡，加上后续因缺粮、感染、传染引起的死亡，人数共计24万多人（存争议）。成为世界历史上最大的水库垮坝惨剧。

美国Discovery频道制作的“10toptechno-logical catastropheinthew orld”专题片，将此次溃坝事件列于首位，居于其后的是印度博帕尔化工厂泄毒事件和前苏联切尔诺贝利核电站爆炸事件。

我采访了受灾最严重的几个村镇，这是他们对那次洪水的回忆。

!

张彦峰《关于洪水的记忆》The Memory of flood

Q：你觉得学院展和非学院展的区别在于哪些方面？

张：要做这个界定恐怕得有很多数据做对比，对于我来说这似乎不重要。

!

张彦峰《关于洪水的记忆》The Memory of flood

多屏录像装置 5′7″，2015

Q：你觉得学院的环境对于艺术家来说是保驾护航还是温水煮青蛙？

张：这个因人而定吧。

!

张彦峰《关于洪水的记忆》The Memory of flood

多屏录像装置 5′7″，2015

Q：你觉得学院里有当代艺术吗？

张：其实对于我来说当代是一个时间概念，我更多思考的是自己做作品，从社会学切入怎么样和社会学保持距离的问题。

!
