**艺术的褶皱**

**The Drapery of Art History**

主持人：郑闻

嘉宾：胡斌 杨好 蒲英玮

地点：南京艺术学院美术馆公共空间

时间：2016年09月14日 15:00

主办：南京艺术学院美术馆

主持人介绍：

```
郑闻，策展人、南京艺术学院美术馆馆长助理与学术部主任、“当代艺术家影像志：隐秘的线索”系列纪录片制片人、“AMNUA 影像档案”策展人，2012 年起就职于南京艺术学院美术馆，负责展览策划、学术研究、馆藏规划，以及美术馆管理。郑闻的策展强调美术馆级别的展览品质，开阔敏锐的艺术视野和鲜明的学术立场。既策划了多个国际现当代大师展览，也不遗余力关注与推广中国青年艺术的发展。
```

```

```

嘉宾介绍：

```
胡斌，广州美术学院美术馆副馆长、艺术管理学系副教授、中国艺术研究院博士。
专注的研究方向有中国革命美术史、中国现当代艺术展览史，长期从事当代艺术的批评与策展。著有《视觉的改造：20世纪中国美术的切面解读》《何以代表“中国”：中国在世博会上的展示与国家形象的呈现》等。
```

```
杨好，青年收藏家,专栏作家,无界艺术Boundless Arts Collection创始人。
毕业于英国圣安德鲁大学和苏富比学院,获艺术史和艺术商业双硕士。师从著名文艺复兴专家 Peter Humfrey ,专注于文艺复兴及欧洲大师画的研究、鉴赏,并由此建立了Boundless Arts Collection的收藏体系。 为上海外语频道《爱收藏》栏目特邀嘉宾;为《三联生活周刊》《文汇报》、 Intelligent Life 中文版、《生活》月刊专栏作家。现在中央美术学院特邀开设西方艺术史课程。
```

```
蒲英玮，艺术家，自由撰稿人，现工作生活于法国里昂。
作品涉及装置、绘画、出版以及写作。以现实社会为背景展开在地性的个人实践。2014年开始对法国大型城市的郊区建设、移民安置以及种族问题展开持续的发掘与呈现。  2013年毕业于四川美术学院油画系现就读于法国里昂国立高等美术学院。2012年获John Moores绘画奖2015年入围IAAC国际艺术评论奖
```
