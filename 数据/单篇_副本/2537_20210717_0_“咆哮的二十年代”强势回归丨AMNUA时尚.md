#“咆哮的二十年代”强势回归丨AMNUA时尚
>作者：艺栢设计	时间：2021/07/17 18:12

>摘要：时尚是一个轮回，没有哪一种风潮会真正地消失。

>阅读：548	点赞：6	在看：7	转发：1	评论：0	赞赏：-100	位置：10

!!**Paco Rabanne 2021FW**

时尚是一个轮回，没有哪一种风潮会真正地消失。在时代的因缘际会之下，时尚风潮总能重新焕发新生。

2021秋冬时装周上，“咆哮的1920年代”风格再次回归，不仅喻示着人们对未来逐渐上升的乐观情绪，还展现了人类超乎想象的秩序重建能力。

!!**Paco Rabanne 2021FW**

提起1920年代，**“爵士时代”、“黄金年代”、“咆哮的二十年代”**等在脑海中瞬间浮现，这些不同的标签组成了1920年代特有的印记。

就像法国作家菲兹杰拉德所言：**“这是一个奇迹的时代，一个艺术的时代，一个挥金如土的时代,也是一个充满嘲讽的时代。”**

时至今日，1920年代的传奇风貌仍然吸引着无数人，那些经典风格也在被不断重温、不断模仿、不断创新。

!!**Paco Rabanne 2021FW**

1920年代是两战之间的和平年代，也是一个持续经济繁荣的时期，更是时装史上起承转合的重要节点。

它所涵盖的激动人心的事件数不胜数，因此还称为“历史上最为多彩的年代”。

**在这期间，新的创造力带领时尚潮流进入新篇章，自由与解放的气息在女性之间蔓延开来，典型的现代主义设计风格开始上显现。**

!**Paco Rabanne 2021FW**

!**Paco Rabanne 2021FW**

纸醉金迷的生活和奢华至极的装饰主义时装，为人们展现了一个华丽的“未来世界”，对“一战创伤”起到强烈的心理安慰作用。

而当下疫情后急需重整的状态，与一战之后百废待兴的世界格局恰好相似。

即使已逾百年，人类社会所面临的问题在本质上依旧没变——往日的生活轨迹被打乱，人们渴望通过一些“努力”来恢复往日的秩序，但是关于后疫情时代的构想又让彷徨无措的人们感到焦虑。

!!!!!**Loewe 2021FW**

**在这种不确定下，许多品牌在本季都暂时把疫情之后成为主流的“舒适性”放下，用亮片、水晶刺绣、珠宝配饰、鲜艳配色和超大廓形以及超短裙摆将“盛装打扮”带回主流视线，暗示着人们对下一个“咆哮的二十年代”到来的渴望。**

!**Loewe 2021FW**

从Prada、Miu Miu、Lanvin、Paco Rabanne、Dries Van Noten，到 Sportmax，再到 Loewe、Rokh 和 Dundas，设计师们集体用1920年风格的廓形和色彩来回应人们对未来的想象。

一种新的能量和渴望正在不断酝酿，亟待爆发。

!!!!!**Saint Laurent 2021FW**

!!!!**Valentino 2021FW**

本季，复古风格的迷你裙再次回归秀场，人们相信，上升变短的裙长预示着未来的乐观情绪。

在经济学领域，裙子的长度常常被当作一个衡量社会经济发展前景的工具。

1926 年，美国经济学家 George Taylor 提出了“裙摆指数”：在经济繁荣的时候，迷你裙会成为主流；而在经济萧条的时候，裙装则趋向保守。

!!!!!!**Dsquared2 2021FW**

新的时代总是带来新的尝试与灵感，被改变的不仅仅是裙摆的长度，还有腰线和材质等。

Saint Laurent 的人造皮草迷你裙、Dsquared2 的金属色迷你裙以及 Valentino 的超短裙造型，都对年轻消费者有着极强的吸引力。

!!!!!**Prada 2021FW**

除了迷你裙之外，亮片、羽毛、皮草、蕾丝、水钻等繁复装饰与色彩、廓形共同组成了五光十色的璀璨光影，更是彰显了强烈的1920年代风格。

Prada 2021秋冬系列，Raf Simons 和 Miuccia Prada 在茧型剪裁的大衣加入皮草滚边装饰，再现了 20 世纪由 Paul Poiret 创造的女性时装廓形；亮片连衣裙和波普艺术风格的印花恰如其分地展现 1920 年代风格的装饰主义精髓。

!!!!!!!**Del Core 2021FW**

不仅如此，设计师们还用大廓形和鲜艳的色彩设计，来满足人们渴望被关注、被看到的心态。

“低调个性”被“张扬外露”所取代，就像 Del Core 秀场那些古怪的夸张造型一样，1920 年代的达达主义与装饰风格被进行融合，完成了向未来主义的演化。

!**Paco Rabanne 2021FW**

当然，1920年代风格的回归，并不仅仅是纸醉金迷，也不意味着我们会突然放弃简单舒适和实用主义的服装风格，它更与时代语境和消费者需求进行了结合，从而呈现出更丰富多元的面貌。

!!**Gucci 2021/2022全新时装系列**

尽管可以预见的‘报复性消费’已经到来，但总体来说人们依旧偏向于选择一些更特别更有个性的设计。

因此从本质上讲，这种变化将十分考验品牌的平衡能力，设计师需要做到兼具舒适性和时髦个性，即在克制和华丽之间找到平衡点。

**结合复古剪裁与大胆配色、时髦美学与舒适性的休闲风格派对服装，将在很长一段时间内大行其道。**

!!!!!!!**Loewe 2021FW**

Loewe 2021 秋冬系列，Jonathan Anderson 就通过极简主义让1920年代风格重获新生。

他用简洁茧型大衣和吊腰连衣裙，结合1920年代的迷幻色彩，为繁复风格注入了符合消费者需求的“极简”版1920年代风格。

!**Loewe 2021FW**

然而，仅仅从时装风格就得出1920年代风格已经回归的结论，显然有点牵强。

一个世纪以前，时装风格的变化是社会、政治和文化背景的变化所引起的生活方式变化的反映。

从工业的空前发展到电影产业的加速发展，从新舞蹈和爵士音乐的蓬勃发再到装饰艺术的鼎盛，许多因素共同促成了那个时代的蓬勃活力，以及在经济和创意上面的繁荣。

!!!**Valentino 2021FW 米兰小剧院**

因此设计师们力求从不同的维度来让1920年代风格完成一次全面完整的回归。

从 Valentino 在米兰小剧院展示新系列，到 Chanel 的 Castel 俱乐部，再到 Loewe 选择在巴黎 Le Train Bleu 餐厅进行展示……

除了时装，他们还将注意力更多地放在了系列的呈现方式上，如音乐、舞蹈以及拍摄场地的选择等，以进一步传达了充满希望的乐观精神。

!**Loewe 2021FW**

**与上世纪二十年代不同的是，“新生的1920年代风格”所代表的不仅是浮夸奢华和纸醉金迷，更多的是一种被选择的结果，符合消费者对未来的美好畅想，并带来积极乐观的情绪。**
