AMNUA学院计划艺术家访谈栏目策划：刘婷

受访者：李伟（以下简称“李”）

!

李伟作品《快-滴-嗒》 kuai-di-da

综合材料 Mixedmedia 1600x1600x300cm，2016

作品阐释：

超出人欲望的无节制消费，是另一种自我囚禁。

Q：你是因为什么进入艺术圈的？

李：我觉得艺术不能以圈来形容。每个艺术家都因该是独立的个体。

!

李伟作品《快-滴-嗒》 kuai-di-da

综合材料 Mixedmedia 1600x1600x300cm，2016

Q：有没有什么因素可以对你的创作思路产生重大影响？

李：对作品产生重大影响的是个人的日常生活状态与经历。

!

李伟作品《本来面目之一》 unmasked.1

装置 Installation 120x120x130cm ，2016

作品阐释：

时间的映射，序破急，如隐秘之花。

Q：作为学院计划的一员，学院的教育或者学院的环境给你带来什么样的影响？

李：学院教育是把双刃剑，用好了如虎添翼，用不好割伤自己。

!

李伟作品《本来面目之一》 unmasked.1

装置 Installation 120x120x130cm ，2016

Q：对于现在常出现的量子力学、生物遗传学等科学技术与艺术的跨界结合，你怎么看？

李：科技不能等同于艺术，科技含量高的未必是好作品，反之亦然。

!

李伟作品《本来面目之一》 unmasked.1

装置 Installation 120x120x130cm ，2016

Q： 能否给我们简单介绍一下你本次参展的作品？

李：我的作品和我的日常体验相关，作品应该如自己的呼吸一般自然，我更关注内省与作品所呈现安静的力量。

!

李伟作品《本来面目之二》 unmasked.2

装置 Installation 120x120x240cm ，2016

作品阐释：

回归生命本质，在时间与空间之间，一期一会。

Q：你觉得学院展和非学院展的区别在于哪些方面？

李：区别在于学院里出格的东西更容易引起讨论和关注。

!

李伟作品《本来面目之二》 unmasked.2

装置 Installation 120x120x240cm ，2016

Q：你觉得学院的环境对于艺术家来说是保驾护航还是温水煮青蛙？

李：学院从某种意义上是正统和保守的代名词，无论在哪，人总是随时受到倒下去的诱惑。

!

李伟作品《本来面目之二》 unmasked.2

装置 Installation 120x120x240cm ，2016

Q： 你觉得学院里有当代艺术吗？

李：当代艺术指正在发生的艺术，是那些关注思想实验和为人类文化提供差异性贡献的视角与实践，重在当下与古老与未来的链接，实验性与不确定性，无论学院内外。

!
