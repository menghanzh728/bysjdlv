### 📊 **数据表**

#### CSV 数据文件（用于Excel/数据分析）

1. **`活动类型年度对比_详细数据.csv`**
   - 格式: 年份 × 活动类型（宽表）
   - 用途: 直接导入Excel做图表/透视表
   - 行数: 14行（2013-2026年）
   - 列数: 8列（年份+7种活动类型+合计）
   - 示例:
     ```
     年份,导览,访谈,讲座,工作坊,公教活动,沙龙,开幕式/研讨会,合计
     2013,0,8,6,0,0,0,0,14
     2014,0,90,58,42,16,12,8,226
     ...
     ```

2. **`活动类型年度对比_百分比.csv`**
   - 格式: 年份 × 活动类型占比百分比（宽表）
   - 用途: 直接查看各年份的活动构成比例
   - 示例:
     ```
     年份,导览%,访谈%,讲座%,工作坊%,公教活动%,沙龙%,开幕式/研讨会%
     2013,0.0,57.1,42.9,0.0,0.0,0.0,0.0
     2014,0.0,39.8,25.7,18.6,7.1,5.3,3.5
     ...
     ```

3. **`活动类型年度对比_长格式.csv`**
   - 格式: 三列（年份、活动类型、数量）- 长表
   - 用途: 数据透视、Python/R数据分析
   - 特点: 每个数据点占一行，便于程序处理
   - 示例:
     ```
     年份,活动类型,数量
     2013,导览,0
     2013,访谈,8
     2013,讲座,6
     ...
     ```

4. **`活动类型总体统计.csv`**
   - 格式: 汇总表（每个活动类型一行）
   - 内容: 总数、占比、最高/最低年份、平均值
   - 用途: 快速查阅各类型的关键指标
   - 示例:
     ```
     活动类型,总数,占比(%),最高年份,最低年份,平均值
     访谈,571,38.5,2015(107),2013(8),40.8
     讲座,455,30.6,2016(85),2022(3),35.0
     ...
     ```
